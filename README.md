## Storage / File System

### Git

Privacy and Security
* `.\.git\logs<<..>>` records/saves all data-assets-related git actions .
* `.\.git\objects\<<..>>` and `.\.git\packed-refs` [d probably is other] use SHA-1 for asset names, that mean relations between copies .
